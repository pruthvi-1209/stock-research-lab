# Stock Research Lab

## Purpose

Stock Research Lab is an engineering research repository dedicated to identifying the most reliable data sources for the Indian stock market.

This repository does not contain the production application.

Its purpose is to answer one question:

> Where should each piece of market data come from?

## Scope

Research only.

No production code.

No UI.

No backend services.

## Research Areas

- Quotes
- Historical Prices
- Company Information
- Financial Statements
- Shareholding Pattern
- Corporate Actions
- News
- Indices
- Exchange Filings
