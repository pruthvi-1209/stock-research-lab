# Research Process

Every research topic follows the same workflow.

1. Define the required data points.
2. Identify candidate providers.
3. Collect evidence.
4. Compare providers.
5. Record trade-offs.
6. Select the preferred provider.
7. Update DATA_SOURCE_DECISION.md.

Rules

- Never assume.
- Never rely on undocumented behaviour.
- Never recommend without evidence.
- Every conclusion must be reproducible.
